# 🤖 포켓몬 검색 봇 (Pokemon Search Bot)

안녕하세요! 트레이너님을 돕기 위해 태어난 **포켓몬 검색 봇**입니다. 삐리찌리릭- ⚡
저는 카카오톡 챗봇 환경을 비롯해 웹 환경, CLI(터미널)에서도 동작할 수 있도록 다양한 모습으로 설계되었어요.
포케토리(Poketory)의 최신 데이터를 기반으로 포켓몬의 평가, 티어 정보, 그리고 결정적인 약점을 빠르고 정확하게 찾아드립니다!

## 🌟 제가 할 수 있는 일 (기능 소개)

저에게 **포켓몬 이름**이나 **타입**(예: 얼음, 드래곤 등)을 말씀해 주시면 다음과 같은 정보를 알아다 드려요!

1. **📖 기본 도감 정보**: 포켓몬의 타입과 배틀에 꼭 필요한 치명적인 약점(4배, 2배)을 콕 집어 설명해 드려요.
2. **🏆 레이드 티어 정보**: 해당 타입에서 몇 티어인지, 추천 스킬과 함께 DPS(초당 데미지)와 TDO(총 누적 데미지)를 분석해 드립니다.
3. **🔰 초보자 추천**: 초보 트레이너분들을 위해 키우기 좋은 포켓몬과 진화 과정, 추천 기술을 안내해 드려요.
4. **🖼️ 포켓몬 공식 이미지**: 오픈소스인 **[PokéAPI](https://pokeapi.co/)**를 활용해서, 검색하신 포켓몬의 공식 아트워크를 아주 선명하게 썸네일로 보여드린답니다! 눈이 즐거우실 거예요. ✨

## 🚀 저를 만나는 방법 (사용법)

### 1. 💬 카카오톡 챗봇 (API 서버)
저는 Flask 기반의 웹 서버(`kakao_bot.py`)로 동작하며 카카오 오픈빌더와 연결해 카카오톡 챗봇으로 활용될 수 있어요.
대화창에 검색어만 툭 던져주시면 텍스트 카드와 이미지 카드를 조합해 깔끔한 답변을 드립니다!

**💻 로컬에서 직접 실행하기**
```bash
# 저를 로컬 개발 환경에서 띄우시려면 이렇게 입력해 주세요! (기본 포트: 5001)
python3 kakao_bot.py
```

**☁️ Render 클라우드 서버에 배포하기 (24시간 가동)**
제가 항상 트레이너님의 부름에 대답하려면 [Render](https://render.com/)와 같은 클라우드 서버에 저를 띄워주시면 좋아요!
저는 클라우드 환경에서 알아서 `PORT`를 감지해 일할 수 있도록 준비되어 있답니다. 
1. Render에서 **New Web Service**를 생성하고 이 저장소를 연결해 주세요.
2. Build Command는 `pip install -r requirements.txt`로 설정해 주세요.
3. Start Command는 `gunicorn kakao_bot:app` (또는 `python3 kakao_bot.py`)로 설정해 주시면 언제든 응답할 준비 완료입니다! 삐리릭- ⚡

### 2. 🌐 웹 브라우저 버전
복잡한 설치 없이 브라우저에서 바로 저를 활용하실 수 있어요.
- 프로젝트 폴더 내 `index.html` 파일을 열고 검색창에 포켓몬이나 타입을 입력해 주세요!

### 3. 💻 CLI (터미널) 버전
터미널이 편하신 개발자 트레이너님들을 위해 스크립트로도 준비되어 있습니다.
```bash
python3 search_pokemon.py [검색어]

# 예시: 
# python3 search_pokemon.py 맘모꾸리
# python3 search_pokemon.py 얼음
```
